# Database Query Fragment Rules

Query fragments in this repository are candidate examples only.

## Contribution Rules

- query fragments are candidates only
- database-specific syntax should be reviewed
- avoid complete commercial search strategy dumps
- avoid copied proprietary database content
- do not include login, cookie, or API-key access details
- do not include bypass or scraping logic

## Scope Guidance

- Keep examples small and reviewable.
- Prefer narrow fragments that show formatting rather than full production search strategies.
- Add notes when a fragment depends on database-specific syntax or field behavior.
