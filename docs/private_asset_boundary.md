# Private Asset Boundary

## Public Repository May Contain

- schemas
- example terms
- example query fragments
- contribution rules
- validation tests
- public documentation

## Private BioMedPilot Resources Retain

- full vocabulary
- core high-value term mappings
- production query builder
- Chinese intent parsing
- AI prompts
- search ranking and scoring
- database routing
- Bioinformatics vs Meta route logic
- commercial search logic
- user query logs
- private feedback data
- premium vocabulary assets

This repository is intentionally limited to collaboration-safe schemas and examples. Public acceptance of an example or candidate file does not imply the same material is used in production, nor does it imply access to BioMedPilot private resources.
